# Templates

### Setup
- Modify `template.py`
- Run `sh init_setup.sh` or  `source init_setup.sh`  

### Notes
此版本是用 venv 當作 virtual environment 管理
- 每次 activate 要用 `source venv/bin/activate`
- 而且他的方式將 modules 放在當下資料夾下
